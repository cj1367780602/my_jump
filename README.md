# Jump Game (跳格子)

这是一个仿微信小程序的 3D 跳格子游戏，使用 Node.js (Express) 和 Three.js 实现。

## 核心玩法
- **蓄力跳跃**: 按住鼠标左键（或长按屏幕）开始蓄力，松开后玩家会根据蓄力时长向前跳跃。
- **得分规则**: 成功落在下一个平台上得 1 分；如果落在平台中心附近，判定为 "Perfect"，额外得 1 分。
- **视角旋转**: 按住鼠标右键可以自由旋转视角，观察平台位置。
- **游戏结束**: 如果玩家没能落在平台上，游戏结束。

## 技术栈
- **后端**: Node.js, Express
- **前端**: Three.js (3D 渲染), OrbitControls (视角控制)

## 快速开始

### 1. 安装依赖
在项目根目录下运行：
```bash
npm install
```

### 2. 启动服务器
```bash
node server.js
```

### 3. 开始游戏
在浏览器中访问：
[http://localhost:3000](http://localhost:3000)

## 操作说明
- **鼠标左键**: 蓄力/跳跃
- **鼠标右键**: 旋转视角
- **鼠标滚轮**: 缩放视角
- **触摸屏**: 长按蓄力跳跃
